# EagleWay Trucking Application Form

Premium web application form for EagleWay Trucking.  
This form is designed for driver applications with automatic payment calculation and all necessary options.

---

## Features

- **Full Name, Phone (numbers only), Email, Nationality**
- **OTR Experience**: Years input
- **Weekly Miles**: Select options (4000, 4500, 5000)
- **ELD Options**: 
  - 4 weeks on | 4 days off  
  - 3 weeks on | 3 days off  
  - 5-6 weeks on | 5-6 days off
- **Payment Box**: Auto-fill CPM and Percent based on OTR experience and Hazmat/Tanker endorsement
- **Endorsements**: TWIC, Hazmat, Tanker (Hazmat only triggers 70 CPM / 30%)
- **Escrow**: $2,500 fully refundable ($250/week)
- **Trailer Type**: Dry Van, Reefer, Both
- **Contact Information**:

  - Email: ziyodaliyevshaxriyor@gmail.com
- **Based Out States**: GA, IL, OH
- **Premium style**: Modern, responsive, mobile-friendly

## Usage

1. Clone or download the repository.
2. Open `index.html` in a browser.
3. Fill in the form fields and submit.
4. Submissions are sent via [FormSubmit](https://formsubmit.co/) to `shaxriyor.z@icloud.com`.

## Files

- `index.html` – Main application form
- `style.css` – Optional separate stylesheet (if separated)
- `script.js` – Payment calculation and endorsement logic

## Notes

- Phone number field accepts **numbers only**.
- Hazmat + Tanker triggers special **70 CPM / 30%** automatically.
- Escrow and ELD are **non-editable**.
- Weekly miles and ELD options are **selectable dropdowns**.
